# **App Name**: NutriSnap

## Core Features:

- Mock Role-Based Auth: Simple authentication mechanism with User and Admin roles for accessing specific app views.
- Wellness Onboarding Flow: Clean metrics intake form to capture height, weight, and age data for personalized tracking.
- Quad-Meal Dashboard: Modular board interface split into four specific meal segments: Breakfast, Lunch, Dinner, and Snacks.
- Smart-Vision Meal Tool: Generative AI powered tool that analyzes either a text description or a food photo to identify nutritional data.
- Activity Logging Log: Interactive chronological feed of tracked intakes with edit capabilities for simple record keeping.
- Modular Data Layer: Architectural scaffolding designed to easily connect Prisma ORM for a robust database transition.

## Style Guidelines:

- Primary color is a Sophisticated Navy (#354F7D) representing professionalism and stability. The background is a Cool Slate Tint (#F5F6F8), and the accent is a Calm Indigo (#7D7DCE) for a contemporary wellness feel.
- Font pairing of 'Literata' (Serif) for headers to provide a premium journal aesthetic, and 'Inter' (Sans-Serif) for dense nutritional data and dashboard labels.
- Minimalist, geometric line-art icons that maintain a lightweight visual profile across meal categories.
- A spacious grid layout utilizing card-based containers with generous white space to prevent cognitive overload.
- Micro-interactions on file uploads and subtle height transitions when expanding meal categories on the dashboard.
